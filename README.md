# 💰 Smart Expense Tracker

A modern web-based expense tracking system built using Flask.

## Features
- Add, edit, delete expenses
- Monthly budget tracking
- Smart insights
- Category-wise charts
- Year-wise analytics

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript
- Chart.js
